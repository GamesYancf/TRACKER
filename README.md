# Tracker de Expediente

Aplicativo PWA simples para rastrear rotas de entregadores, calcular distância, tempo e custos de combustível.

## Como usar

1. Abra `index.html` no navegador ou execute um servidor local.
2. Clique em **Iniciar Expediente** e permita o acesso à localização.
3. O mapa vai desenhar a rota em tempo real e as estatísticas serão atualizadas.
4. Clique em **Encerrar Expediente** ao terminar para salvar o relatório no histórico.

## Recursos

- Registro de pontos GPS em tempo real
- Cálculo de distância percorrida
- Relatório diário com data, horário, tempo total e consumo de combustível
- Histórico local armazenado em `localStorage`
- PWA instalável e cache para funcionamento offline do app shell
